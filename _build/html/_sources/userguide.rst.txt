USER GUIDE
=============
Getting started.
===============
Once you have logged in the dhis2 instance,in the Apps Menu click the metadata audit app.This will then direct to a landing page that looks like this.
This page displays the types of metadata in dhis2 that one would like to track.
It also shows two highlights;
1.The recent highlight.
This shows the most latest updates perfomed on the metadata,the time it was edited and the user that made the changes.
2.The popular highlight.
This shows the most edited metadata,the number of edits made and the user that made the changes.
